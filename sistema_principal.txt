Sistema principal:
    carrega encodings e ids reconhecidos - OK

    inicializa a câmera - 
    carrega imagem da câmera - 

    detecta faces na câmera
    verifica se tem faces
    verifica se a face está próxima o suficiente da câmera

    codifica a face mais próxima da câmera X vezes
    
    compara face codificada com lista de codificações carregado
    se encontrar, envia Notificação com id

    notifica banco de dados que aluno entrou
    banco de dados mostra o nome do aluno que entrou

    aluno recebe retorno visual que o reconhecimento funcionou

    recomeça

Cadastro:
    Digita o id do aluno (número ou matrícula, podemos gerar automático?)
    Recebe lista de 1 ou mais fotos do rosto do aluno
    
    cria pasta no path padrão, salva imagens lá
    se puder, cada imagem é carregada e codificada 100 vezes ou mais
    salva arquivo .enc com a codificação feita, se não houver

    se puder, adiciona ID e codificação á lista já utilizada
    sucesso

Carregar rostos:
    Recebe pasta padrão com todos os cadastrados
    para cada pasta com os IDs, analisar os arquivos dentro

    tentar carregar arquivo .enc, se não der ignora
    usar ID(nome da pasta) e codificação carregada para salvar no código
    
    repitir com todas as pastas e arquivos
    sucesso
